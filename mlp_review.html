Multi-Layer Perception & Micrograd

Multi-Layer Perceptron (MLP)

Neuron (뉴런)

MLP의 기본 연산 단위. 생물학적 뉴런을 수학적으로 모델링함.

  • 여러 입력값 x에 가중치 w를 각각 곱함
  • 가중합(weighted sum)에 편향(bias) 가산
  • 활성화 함수를 통해 최종 출력값 결정
  • 대표 활성화 함수: Sigmoid, ReLU, Tanh

학습 메커니즘

가중치 최적화를 위해 아래 과정 반복.

  • 순전파: 입력층 → 은닉층 → 출력층 순 계산, 예측값 생성
  • 손실 함수: 예측값·실제값 오차 수치화 (MSE, Cross-Entropy)
  • 역전파: 오차를 출력층 → 입력층 방향 전달, 기울기 계산
  • 경사하강법: 기울기로 가중치 업데이트, 손실 최소화

Micrograd

Gradient (기울기)

어떤 값이 출력(Loss)에 미치는 영향의 크기. dL/dx 로 표현하며, 값이 클수록 x의 미소 변화가 Loss에 큰 영향을 줌. 

class Value:
    def __init__(self, data):
        self.data = data
        self.grad = 0.0        # dL/d(self), 초기값 0
        self._backward = lambda: None

    def __mul__(self, other):
        out = Value(self.data * other.data)

        def _backward():
            # d(out)/d(self) = other.data
            self.grad  += other.data * out.grad
            other.grad += self.data  * out.grad

        out._backward = _backward
        return out

Chain Rule (연쇄 법칙)

여러 연산이 연결될 때 각 단계의 기울기를 곱하여 전체 기울기 산출.

L = f(g(x)) 일 때  →  dL/dx = (dL/df) × (df/dg) × (dg/dx) 

# z = x * w,  L = z * b
x = Value(2.0)
w = Value(3.0)
b = Value(4.0)

z = x * w       # z.data = 6.0
L = z * b       # L.data = 24.0

L.grad = 1.0   # 출력 기준점 설정
L._backward()   # dL/dz = b = 4.0,  dL/db = z = 6.0
z._backward()   # dL/dx = dL/dz * w = 4 * 3 = 12.0
               # dL/dw = dL/dz * x = 4 * 2 = 8.0

계산 그래프 시각화 — L = (x × w) × b
